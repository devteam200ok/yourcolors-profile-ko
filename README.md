# Your Colors

**Your Colors**는 디지털 텍스타일 프린팅 라이선싱 & 생산 플랫폼입니다.  
70년 전통의 섬유 기업인 **[Taihan Textile Co., Ltd.]** (주식회사 대한방직)이 운영하며, 디자이너가 패턴 디자인을 올리고 라이선싱 수익을 얻을 수 있도록 지원합니다.  
업로드된 패턴은 대한방직 대구 공장에서 고품질 원단으로 인쇄, 가공, 롤링되어 전 세계에 출고됩니다.

## 🧵 플랫폼 개요 — 우리가 하는 일

- 디자이너가 패턴 디자인(디지털 파일)을 플랫폼에 업로드  
- 해당 디자인에 대해 라이선스를 부여하고, 디자이너는 라이선싱 수익 획득  
- 브랜드나 제조사에서 주문 시, 대한방직 대구 공장에서 원단 인쇄 → 가공 → 완제품 원단으로 생산  
- 전 세계 고객에게 고품질 커스텀 원단을 공급  
- 특히 **소량 주문, 빠른 납기, 높은 품질**을 원하는 브랜드 및 제조사에 최적화

## 🏭 디지털 텍스타일 프린트 방식 및 강점

우리의 핵심은 **반응성 염료(reactive dye) 기반 디지털 프린트**입니다.  
면(cotton), 린넨(linen) 등 자연섬유 원단을 사전 처리하고,  
디자인을 잉크젯으로 인쇄한 뒤, 스팀/고온 고정, 세척, 텐터(-건조/신장), 산포라이징, 캘렌더링 등의 후가공을 거칩니다.  
이 과정을 통해 마치 전통적인 스크린 또는 로터리 프린트와 동일한 품질 수준을 유지하며,  
색상은 섬유에 화학적으로 결합되어 세탁성과 내구성, 부드러운 촉감이 뛰어납니다.

**기존 전사 프린트 또는 안료 기반 디지털 프린트와의 차별점**  
- 단순 표면 코팅이 아닌, 섬유 내부에 염료가 결합 → 세탁 및 마모에 강함  
- 부드러운 핸드감 유지  
- 선명하고 다채로운 색상 구현 가능

## 🎯 주요 고객층 (타깃 사용자)

- 패션·의류 브랜드, 홈텍스타일 브랜드 등 — **커스텀 패턴 + 소량 또는 중량 주문**이 필요한 경우  
- 프리랜서 디자이너, 패턴 아티스트 — **자신의 디자인을 라이선싱하여 수익화**하려는 분들  
- 스타트업, 소규모 브랜드 — **대량 주문 부담 없이 소규모 캡슐 컬렉션, 시즌 한정, 샘플 주문**이 필요한 경우  
- 한정판, 리미티드 에디션, 빠른 회전이 필요한 컬렉션 등 — **납기와 유연성이 중요한 브랜드**

## 📦 생산 및 주문 워크플로우 (예시)

1. 디자이너가 Seamless 패턴 또는 플랫 디자인을 플랫폼에 업로드  
2. 고객(브랜드/제조사)이 디자인, 원단 종류(면, 린넨 등), 주문 수량 선택 — 최소 1야드부터 주문 가능  
3. 주문 접수 → 대한방직 대구 공장에서 디지털 프린트 + 후가공 처리 (염료 인쇄 → 고정 → 세척 → 텐터/캘렌더 → 품질 검사 → 롤링)  
4. 완성된 원단을 롤링 및 포장 → 고객사에 글로벌 출고  
5. 디자이너는 해당 주문 건에 대해 라이선싱 로열티 수익을 획득
